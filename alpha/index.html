Alpha Description: Trader Joe's Microinteraction

Original Interface: Trader Joe's - From the Freezer

Triggers:

  • Object: "+ ADD TO LIST" button centered at the bottom of a product's card
    • Hover on
    • Hover off
    • Click

Rules:

  • Object: "+ ADD TO LIST" button
    • Hover on:
      • Curser changes to finger
      • Drop shadow grows (about 20% opacity)
      • Button boarder weight thickens (about 3px)
      • Button color fills with beige
    • Hover off:
      • Curser changes back to default arrow
      • Drop shadow shrinks back to default
      • Button boarder weight thins back to default
      • Button color fills back to default white
    • Click:
      • Button gets wider
      • + icon switches to check mark
      • Text switches to "ADDED TO LIST"
      • Border color fills with red
      • Button boarder weight thickens (about 3px)
      • Item is added to internal shopping list database (and interface on different page)
